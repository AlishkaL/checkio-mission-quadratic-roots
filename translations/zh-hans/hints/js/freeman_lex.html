如何完成代码？

唯一剩下的情况（当discriminant < 0 时）要求将某个字符串作为数组的单个元素返回。 

return ["No real roots"]
那么，我的最终解决方案应该是怎样的呢？

您只需将之前所有的提示连接起来，然后将代码封装成函数表达式即可。强烈建议对输入和输出数据使用类型提示。

下面是整个解决方案：

function quadraticRoots(a: number, b: number, c: number): [number, number] | [string] {
    
    const discriminant = b * b - 4 * a * c;
    if (discriminant> 0) {
        const root1 = (-b + Math.sqrt(discriminant)) / (2 * a);
        const root2 = (-b - Math.sqrt(discriminant)) / (2 * a);
        return [root1, root2]}
    if (!discriminant) {
        const root = -b / (2 * a);
        return [root, root]}
    return ["No real roots"]
}
