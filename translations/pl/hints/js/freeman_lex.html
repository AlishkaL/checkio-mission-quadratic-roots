Nie mam pojęcia, jak zacząć rozwiązywać tę misję.

Proszę złapać pomysł 💡! Proszę postępować zgodnie z algorytmem matematycznym. Proszę obliczyć wyróżnik, sprawdzić czy jest większy od 0 - obliczyć i zwrócić dwa pierwiastki, jeśli jest równy 0 - obliczyć i zwrócić jeden pierwiastek. W przeciwnym razie proszę zwrócić ciąg znaków.

Ok, od czego powinienem zacząć?

Proszę obliczyć dyskryminantę. 

const discriminant = b * b - 4 * a * c;
Co robić dalej?

Proszę użyć wyrażenia if. Ponieważ każda instrukcja if kończy się instrukcją return, nie muszą Państwo używać słów kluczowych else if lub else. Najpierw proszę utworzyć gałąź, w której discriminant> 0. 

if (discriminant> 0) {
    ...}

Proszę utworzyć wyrażenie obliczające roo1, root2 zgodnie z formułą ... 

const root1 = (-b + Math.sqrt(discriminant)) / (2 * a);
const root2 = (-b - Math.sqrt(discriminant)) / (2 * a);

... i zwrócić je jako tablicę. 

return [root1, root2]
Nadal potrzebuję podpowiedzi.

Proszę sprawdzić drugi przypadek, gdy discriminant == 0 lub !discriminant. W tym przypadku proszę obliczyć tylko jedną wartość root i zwrócić tablicę z dwiema takimi samymi wartościami. 

if (!discriminant) {
    const root = -b / (2 * a);
    return [root, root]}
Jak zakończyć kod?

Jedyny przypadek (gdy discriminant < 0) wymaga zwrócenia określonego ciągu jako pojedynczego elementu tablicy. 

return ["No real roots"]
Jak więc powinno wyglądać moje ostateczne rozwiązanie?

Wystarczy połączyć wszystkie poprzednie podpowiedzi i zawinąć kod w wyrażenie funkcyjne. Używanie podpowiedzi typu dla danych wejściowych i wyjściowych jest wysoce zalecane.

Oto całe rozwiązanie:

function quadraticRoots(a: number, b: number, c: number): [number, number] | [string] {
    
    const discriminant = b * b - 4 * a * c;
    if (discriminant> 0) {
        const root1 = (-b + Math.sqrt(discriminant)) / (2 * a);
        const root2 = (-b - Math.sqrt(discriminant)) / (2 * a);
        return [root1, root2]}
    if (!discriminant) {
        const root = -b / (2 * a);
        return [root, root]}
    return ["No real roots"]
}
