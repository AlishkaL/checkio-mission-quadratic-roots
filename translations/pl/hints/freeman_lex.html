Proszę złapać pomysł 💡! Proszę postępować zgodnie z algorytmem matematycznym. Proszę obliczyć wyróżnik, sprawdzić czy jest większy od 0 - obliczyć i zwrócić dwa pierwiastki, jeśli jest równy 0 - obliczyć i zwrócić jeden pierwiastek. W przeciwnym razie proszę zwrócić ciąg znaków.

Ok, od czego powinienem zacząć?

Proszę obliczyć dyskryminantę. 

discriminant = b * b - 4 * a * c
Co robić dalej?

Proszę użyć wyrażenia if. Ponieważ każde if kończy się instrukcją return, nie muszą Państwo używać słów kluczowych elif lub else. Najpierw proszę utworzyć gałąź, w której discriminant> 0. 

if discriminant> 0:
    ...

Należy zaimportować moduł math, aby użyć jego funkcji sqrt. Proszę utworzyć wyrażenie obliczające roo1, root2 zgodnie z formułą ... 

root1 = (-b + math.sqrt(discriminant)) / (2 * a)
root2 = (-b - math.sqrt(discriminant)) / (2 * a)

... i zwrócić je jako listę. 

return [root1, root2]
Nadal potrzebuję podpowiedzi.

Proszę sprawdzić drugi przypadek, gdy discriminant == 0 lub not discriminant. W tym przypadku proszę obliczyć tylko jedną wartość root i zwrócić listę z dwiema takimi samymi wartościami. 

if not discriminant:
    root = -b / (2 * a)
    return [root, root]
Jak zakończyć kod?

Jedyny przypadek (gdy discriminant < 0) wymaga zwrócenia określonego ciągu jako pojedynczego elementu listy. 

return ["No real roots"]
Jak więc powinno wyglądać moje ostateczne rozwiązanie?

Wystarczy połączyć wszystkie poprzednie podpowiedzi i zawinąć kod w wyrażenie funkcyjne. Używanie podpowiedzi typu dla danych wejściowych i wyjściowych jest wysoce zalecane.

Poniżej znajduje się całe rozwiązanie:

from collections.abc import Iterable
import math


def quadratic_roots(a: int, b: int, c: int) -> Iterable[int|str]:
    
    discriminant = b * b - 4 * a * c
    if discriminant> 0:
        root1 = (-b + math.sqrt(discriminant)) / (2 * a)
        root2 = (-b - math.sqrt(discriminant)) / (2 * a)
        return [root1, root2]
    if not discriminant:
        root = -b / (2 * a)
        return [root, root]
    return ["No real roots"]

Jasne, proszę łapać!

W tym rozwiązaniu zwraca Pan obiekt Iterable, a nie listę. Proszę użyć try/except, aby obsłużyć przypadek, gdy discriminant < 0, a następnie zwrócić pierwszy pierwiastek, obliczyć i ponownie zwrócić drugi pierwiastek pierwszego pierwiastka. 

from collections.abc import Iterable
import math


def quadratic_roots(a: int, b: int, c: int) -> Iterable[int|str]:
    
    try:
        d_sq = math.sqrt(b * b - 4 * a * c)
        yield (root1:= (-b + d_sq) / (2 * a))
        yield d_sq and (-b - d_sq) / (2 * a) or root1
    except: yield "No real roots"
