Ok, what should I start from?

Let's calculate discriminant. 

const discriminant = b * b - 4 * a * c;
What to do next?

Use if statement. Since every if ends with return statement, you actually don't need to use else if or else keywords. First, built a branch where discriminant > 0. 

if (discriminant > 0) {
    ...}

Create expressions to calculate roo1, root2 according to the formula ... 

const root1 = (-b + Math.sqrt(discriminant)) / (2 * a);
const root2 = (-b - Math.sqrt(discriminant)) / (2 * a);

... and return them as an array. 

return [root1, root2]
I still need a hint.

Check the second case, when discriminant == 0 or !discriminant. In this case calculate only one root and return an array with two same values. 

if (!discriminant) {
    const root = -b / (2 * a);
    return [root, root]}
How to finish the code?

The only case left (when discriminant < 0) demands returning of a certain string as a single element of an array. 

return ["No real roots"]
So, how my final solution should look like?

You just need to join all previous hints and wrap your code into function expression. Using type hints for input and output data are highly recommended.

Here is the whole solution:

function quadraticRoots(a: number, b: number, c: number): [number, number] | [string] {
    
    const discriminant = b * b - 4 * a * c;
    if (discriminant > 0) {
        const root1 = (-b + Math.sqrt(discriminant)) / (2 * a);
        const root2 = (-b - Math.sqrt(discriminant)) / (2 * a);
        return [root1, root2]}
    if (!discriminant) {
        const root = -b / (2 * a);
        return [root, root]}
    return ["No real roots"]
}
